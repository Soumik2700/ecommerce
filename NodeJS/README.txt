==> Make the clone of the project by using "git clone https://github.com/Soumik2700/ecommerce.git"

==> open the terminal and go in the project by using cd <folder name>

==> run the command npm install as the node_modules are not includes with the project. It will install all the dependencies used in the project

==> run "npm start" in the terminal to start the server and to establish connection with mongodb server.

==> test all the CRUD operation with Postman or Thunder Client.
